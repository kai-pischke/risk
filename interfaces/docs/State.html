Description

The basic representation of a Game excluding all dynamic and non-deterministic elements. This module focuses only on encapsulating all information needed to reconstruct a given game position and doesn't focus on the actual mechanics or logic of gameplay.

changeMiniPhase :: MiniPhase -> GameState -> GameState Source #

Does nothing if not in Attack Phase, otherwise sets phase to Attack MiniPhase (inserting the provided MiniPhase). Note that calling this function on a WonBattle will cause it to remember that the current player gets a card this turn.

useCard :: Player -> Card -> GameState -> GameState Source #

Puts the card in the discard pile.

drawCard :: Player -> GameState -> GameState Source #

Adds the top card to the player's hand. Shuffles the discard pile if necessary using a provided shuffle function.

kick :: Player -> Player -> GameState -> GameState Source #

Removes the Player given in the second argument from the game. Gives all cards to the Player given in the first argument.

hasDrawn :: GameState -> Bool Source #

True if and only if the current player has drawn a card this turn.
